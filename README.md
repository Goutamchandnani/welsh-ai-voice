# Llais Cymraeg (Welsh Voice)

An open-source, developer-first Welsh language voice AI stack.

## Project Structure
- `api/`: FastAPI & WebSocket streaming gateway.
- `data/`: Raw and processed datasets (Mozilla Common Voice).
- `docs/`: Technical documentation and integration guides.
- `models/`: Training scripts and model configuration.
- `notebooks/`: Research and development (Kaggle/Colab).
- `scripts/`: Utility scripts for data and deployment.
